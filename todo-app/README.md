# Todo App (Spring Boot + React)

## Backend (Spring Boot + MySQL)
1. Install MySQL and create database `tododb`
2. Update `application.properties` with your MySQL username/password
3. Run backend:
   ```
   cd backend
   mvn spring-boot:run
   ```

## Frontend (React)
1. Run:
   ```
   cd frontend
   npm install
   npm start
   ```

## Features
- Register and Login
- Add and List Todos
